دورة HTML

HTML هي طريقة لكتابة وهيكلة صفحات الويب ، ولا تعتبر لغة برمجية بالمعنى الفعلي بل هي لغة وصفية أي انها تستخدم لوصف الصفحة وتوزيع وترتيب محتواها تستخدم Tags وهي مايسمى بالوسوم ...

اهداف الدورة

  • التعرف على مفهوم الHTML
  • تعلم كل tag والغرض من استخدامه
  • طرق اضافة النصوص، الفيديوهات وغيرها من المحتويات المعتادة في صفحة الويب

متطلبات الدورة

  • متصفح (سفاري، قوقل كروم، ...)
  • محرر نصوص
